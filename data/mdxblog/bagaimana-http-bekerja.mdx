Saat kalian membaca ini besar kemungkinan bahwa kalian menggunakan http untuk menerima informasi yang
didapatkan melalui server lalu ditampilkan pada layar browser. HTTP atau Hyper Text Trasnfer Protocol
merupakan Protocol untuk bertukar informasi antara klien dengan server. Yang mana klien ini adalah
browser yang meminta informasi kepada server dan server akan mengolah permintaan dan merespon kembali
ke browser lalu browser menerima respon berbentuk html atau json tergantung dari type data di http header.
