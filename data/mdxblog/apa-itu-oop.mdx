Object Oriented Programming adalah sebuah paradigma programming yang mengatur bagaimana sebuah program yang
memiliki struktur dan tingkah yang sama kedalam sebuah objek. Objek hampir sama seperti array atau dictionary
di python, objek juga dapat menyimpan data ke dalam properti tapi yang membedakannya adalah juga memiliki method
yang mendekripsikan objek tersebut.
## Bagaimana cara Membuat sebuah Objek di Python
Objek di mulai dengan keyword class dan nama objek seperti dibawah ini

```
class Ninja:
    chakra = 10
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

    def __init__(self):
        pass
