Karena aku telah mempelajari coding kupikir ide bagus untuk membuat website untuk menampilkan tulisanku, jadi kuputuskan untuk membuat blog.

Aku telah memutuskan menggunakan next js dan tailwind untuk style. Pertama aku berpikir untuk menyimpan post di json file tapi semakin banyak post yang kumiliki bakal semakin sulit untuk merubah post sebelumnya jadi aku mencari di google bagaimana cara membuat blog dan menemukan post ini. 

Setelah membacanya, aku merasa yakin untuk memakai mdx. Entah kenapa setiap kali mencoba membuat website aku selalu merasa kurang puas dengan desain.

Masih banyak fitur yang yang ingin kutambah seperti mencari post menggunakan title atau mengolompokkan post berdasarkan kategori.

Website ini memakan banyak waktu dalam pembuatan terdapat beberapa bug terutama saat mencoba menggunakan mdx dengan next js. Menggunakan mdx dengan astro js jauh lebih mudah daripada next js dan mungkin kedepan akan kuganti dengan astro js tapi sekarang akan tetap menggunakan next js.

Terdapat kepuasan tersendiri dalam membuat website karena aku hanya mengandalkan dokumentasi saja, dan dalam proses dapat banyak belajar.

Masih banyak yang ingin kuperbaiki dari segala sisi baik dari website ini sendiri maupun kualitas tulisan.
